DNS-Einrichtung – Einmalig, fur alle

Nach dieser Einrichtung funktioniert JDS Mail fur alle Nutzer – egal ob joel@, info@, support@jds-search.de.

Schritt 1: Server-IP herausfinden

Dein Server ist der Rechner, auf dem JDS lauft (VPS, Root-Server, Hetzner, Netcup, Contabo...).

Methode 1: Beim Hoster nachschauen
Logge dich bei deinem Hoster ein. Die IP steht direkt im Dashboard neben deinem Server. Z.B. 49.12.34.56
Methode 2: Per SSH-Befehl
curl -4 ifconfig.me
Gibt direkt deine öffentliche IPv4 aus, z.B. 49.12.34.56
Methode 3: Lokale Entwicklung
Fur lokale Tests brauchst du keine DNS-Eintrage. Mail funktioniert intern per check_mail-Befehl. Siehe Schritt 4.
Schritt 2: DNS-Eintrage setzen (2 Stuck)

Bei deinem Domain-Provider (Cloudflare, Ionos, Strato, Hetzner DNS-Console)

Eintrag 1 – Mails empfangen
TypMX
Host / Name@ (oder leer lassen)
Wert / Zielmail.jds-search.de
Prioritat10
Eintrag 2 – Mail-Server erreichbar machen
TypA
Host / Namemail
Wert / ZielDEINE.SERVER.IP (aus Schritt 1)

Optional fur Spam-Schutz:

Eintrag 3 (optional) – Spam vermeiden
TypTXT
Host / Name@
Wertv=spf1 mx a ~all
Schritt 3: Catch-All-Weiterleitung einrichten

Damit ALLE @jds-search.de-Adressen automatisch ankommen:

Bei deinem Domain-Provider / E-Mail-Provider:
Catch-All aktivieren:
Alle E-Mails an *@jds-search.de
weiterleiten an catchall-jds@gmail.com
Schritt 4: JDS Mail-Poller starten (ein Befehl)

Diese eine Zeile holt alle Mails ab und verteilt sie automatisch an die richtigen JDS-Nutzer:

python manage.py check_mail --user catchall-jds@gmail.com --password DEIN_APP_PASSWORT
Wie es funktioniert:
1. GMX schickt Mail an joel@jds-search.de
2. Catch-All leitet sie an catchall-jds@gmail.com
3. Der Poller holt sie per IMAP ab
4. Erkennt joel@ → sucht Nutzer joel
5. Mail erscheint in Joels JDS-Posteingang

Fertig. Funktioniert fur alle Nutzer. Einmal einrichten.
Gmail App-Passwort erstellen
1. Google Sicherheitseinstellungen
2. "2-Faktor-Authentifizierung" aktivieren
3. Danach: "App-Passworter" → Neues Passwort
4. Name: "JDS Mail" → 16-stelliges Passwort kopieren
5. In den Befehl oben einfugen

Das war's. Jetzt kann jeder eine @jds-search.de-Adresse nutzen. JDS Mail ist bereit.

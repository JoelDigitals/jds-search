JDS Mail einrichten – 3 Schritte

Nach dieser Einrichtung erhalten alle @jds-search.de-Adressen echte Mails von GMX, Gmail, Web.de & Co.

Schritt 1: ForwardEmail.net einrichten (2 min)

ForwardEmail.net ist kostenlos fur Custom-Domain-Mail-Weiterleitung.

1. Account erstellen auf forwardemail.net
2. Domain hinzufugen: jds-search.de
3. Domain verifizieren (per TXT-Record – wird dir angezeigt)
4. Aliase anlegen:
   *@jds-search.de → Webhook https://DEIN-SERVER/mail/webhook/

Fertig. Jede Mail an irgendwas@jds-search.de wird per POST an deinen JDS-Server geschickt.
Schritt 2: JDS-Server bereitstellen

Dein Django-Server muss von außen erreichbar sein:

Render.com: https://jds-search.onrender.com/mail/webhook/
Eigener Server: https://jds-search.de/mail/webhook/

Der Webhook-Endpunkt ist bereits eingebaut unter /mail/webhook/
Er empfangt POST-Requests mit from, to, subject, text und stellt Mails automatisch zu.
Schritt 3: Testen
Lokal testen:
curl -X POST http://localhost:8000/mail/webhook/ -d "from=test@gmx.de" -d "to=JoelDigitals@jds-search.de" -d "subject=Test" -d "text=Hallo"
Erwartete Antwort:
{"status": "ok"}
Die Mail erscheint dann sofort im Posteingang von JoelDigitals.
So funktioniert's im Betrieb
GMX-Nutzer schreibt an joel@jds-search.de
→ ForwardEmail.net empfangt die Mail
→ Leitet sie per POST an /mail/webhook/
→ JDS erkennt joel → sucht Nutzer "joel"
→ Mail erscheint in Joels JDS-Posteingang

Funktioniert fur ALLE Nutzer. Einmal eingerichtet, nie wieder anfassen.
